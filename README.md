# 🎙️ Voice Fraud Detection System

AI-based system to detect deepfake / spoofed audio using machine learning.

---

## 🚨 Problem
With the rise of AI-generated voices, it has become easy to create fake audio clips that mimic real people. This poses serious risks in fraud, misinformation, and security.


## 💡 Solution
We propose a voice forensics system that:
- Extracts audio features (MFCC, spectrogram)
- Uses deep learning (CNN/LSTM) for classification
- Detects whether an audio is **Real or Fake**


## ⚙️ Pipeline

Audio Input → Feature Extraction → Model Prediction → Result
